# 開発引き継ぎドキュメント (Handoff)

## 1. セッション終了時のステータス
- **Webデプロイ成功**: GitHub Actions により、Push毎に自動でGitHub Pagesへデプロイされる環境が整いました。
- **実機確認済み**: iPhone等のスマホブラウザで、移動・戦闘・採取・デバッグメニュー・シーン遷移が動作することを確認しました。
- **コードクリーンアップ**: 開発用のプリントログを削除し、仕様に合わせてコメントを整備しました。
- **技術仕様書**: `antigravity-prototype-toh/docs/specifications.md` に最新の仕様をまとめています。

## 2. 実装済みの重要機能
- **自動戦闘・採取**: プレイヤーが立ち止まると、敵がいれば攻撃、いなければ採取を自動で行います。
- **拠点拡張**: 集めた素材を使って、攻撃力、パーティ上限、バッグ容量をアップグレード可能です。
- **デバッグツール**: 三点タップ、またはデバッグ用ボタンからメニューを開き、各種ギズモ（当たり判定や枠線）を表示できます。
- **昼夜サイクル**: 時間経過で空の色が変わり、夜間は敵のスポーン速度が2倍になります。

## 3. 次回への申し送り事項
- **アセットのブラッシュアップ**: 現在は仮素材（プレースホルダー）を使用しています。ビジュアルを強化する際は `assets/placeholders/` 内の画像を差し替えてください。
- **ビルド環境の注意**: 
  - GitHub Actions では **Godot 4.3 Standard (非.NET版)** を使用しています。
  - プロジェクト設定で `physics/3d/physics_engine` は Godot のデフォルトを使用しています（Jolt Physics はCI環境での互換性のため一旦外しました）。
- **デバッグ操作**: 
  - 実機テスト時は「画面を三本指で1.5秒間長押し（三点タップ相当）」でデバッグメニューが開きます。

## 4. プロジェクト構造
- `.github/workflows/deploy.yml`: 自動デプロイ設定（リポジトリルート）
- `antigravity-prototype-toh/`: Godotプロジェクトルート
- `docs/specifications.md`: 技術詳細ドキュメント
