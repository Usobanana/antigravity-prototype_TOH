# 技術仕様書 (v1.0)

## 1. プロジェクト概要
本作は、拠点の設営（Base）とフィールドでの自動戦闘・採取（Field）を繰り返す、アクションアドベンチャーゲームです。

### 開発環境
- **エンジン**: Godot 4.3 Stable
- **スクリプト**: GDScript
- **ターゲット**: Web (iPhone/Android ブラウザ対応)

---

## 2. コアシステム

### 昼夜の概念 (TimeManager)
- **サイクル**: 実時間で約1分で1日が経過。
- **環境変化**: 
  - `TimeManager` が `time_changed` シグナルを発火。
  - `DirectionalLight3D` の角度と色が動的に変化（朝・昼・夕・夜）。
- **夜間ボーナス**: 夜間は敵のスポーン率が **2倍** に向上。

### リソース・永続化 (GameStateManager)
- **管理データ**: 木材、石材、鉄鉱石、所持上限（Hold）、拠点レベル等。
- **ストレージ**: プレイヤーのバッグ（Hold）には上限があり、拠点に帰還すると自動的に拠点倉庫へ転送される。

---

## 3. ゲームプレイ・キャラクター

### プレイヤー (Player)
- **移動**: バーチャルパッドによる 360度移動。
- **自動行動**: 
  - 範囲内の敵を検出し、優先的に攻撃。
  - 周辺に敵がいない場合、リソース（木・岩）の採取を開始。
- **スキル**: 攻撃を当てることでゲージが溜まり、アイコンタップで必殺技を発動可能。

### パーティ (Allies)
- **フォロワー**: 最大パーティ人数（GameCenter で管理）まで雇用可能。
- **追従アルゴリズム**: プレイヤーの後ろを一定距離で追従し、プレイヤーが攻撃している対象を共に攻撃。

### エネミー (Enemies)
- **通常種**: 近接攻撃を行う。
- **高速種 (FastEnemy)**: 移動速度が速い。
- **遠距離種 (RangedEnemy)**: 離れた場所から弾を飛ばす。
- **ボス (BossEnemy)**: 高耐久・高攻撃力の大型エネミー。

---

## 4. UI/デバッグ機能

### デバッグメニュー (DebugMenu)
- **開閉方法**: プレイ中に画面内を同時に 1.5秒間 **三点タップ** することで表示。
- **機能**:
  - リソース（木・石・鉄）の即時追加。
  - レベル（攻撃力、パーティ上限、バッグ容量）の即時アップ。
  - **ギズモ表示**: グリッド、アセット枠、当たり判定の表示ON/OFF。

---

## 5. 技術アーキテクチャ

### シーン構造
- `World.tscn`: ルートシーン。HUD、デバッグメニュー、現在のマップを包含。
- `Base.tscn`: 拠点シーン。
- `Field.tscn`: 戦闘フィールド。
- `NavigationRegion3D`: 敵や仲間の移動計算に使用。

### ビルド・デプロイ
- **GitHub Actions**: `deploy.yml` により、Push時に自動でWeb版をビルド・デプロイ。
- **SharedArrayBuffer対応**: GitHub Pages上での動作を安定させるため、`coi-serviceworker.js` を自動注入。
