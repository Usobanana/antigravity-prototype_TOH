# 開発用語集 (Glossary)

このプロジェクトで使用する用語や命名規則の定義まとめ。

## UI・システム用語
- **採取ゲージ (Gather Progress)**: プレイヤーが採取可能なオブジェクト（木や石など）の周囲に立ち止まった際に開始される、採取完了までの時間を表すプログレスバー。

## ゲーム内リソース
- **Wood (木)**: Tree（木オブジェクト）から取得できる基本素材。拠点の施設アップグレード等に使用。
- **Stone (石)**: Stone（岩オブジェクト）から取得できる基本素材。施設のアップグレードにWoodと共に使用。

## 状態 (State)
- **IDLE (停止)**: プレイヤーが立ち止まっている状態。この時のみ索敵・採取判定が行われる。
- **MOVE (移動)**: バーチャルパッド等で移動中の状態。攻撃・採取よりも優先され、実行中は他のアクションをキャンセルする。
- **ATTACK (攻撃)**: 自動戦闘中の状態。
- **GATHER (採取)**: 自動採取中の状態。

## その他
- **Base (拠点)**: 安全圏。フィールドで集めた素材を使って「Tent」や「Camp」等の施設をアップグレードし、パーティ上限やステータスを上げる場所。
- **Field (フィールド)**: 敵や素材が配置された主要な探索エリア。
